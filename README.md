# China Trade Platform

Professional China Trade & Logistics Platform for Middle East Markets

## Overview

A comprehensive end-to-end solution for China trade, serving Middle East markets with complete services including product sourcing, quality inspection, warehousing, shipping, customs clearance, and logistics solutions.

## Features

### Core Services
- **Product Sourcing**: Find the best suppliers and products in China with expert sourcing team
- **Quality Inspection**: Ensure product quality with thorough inspections and factory audits
- **Warehousing**: Secure storage solutions in strategic locations across China
- **Shipping & Freight**: Air, sea, rail, and trucking options for all shipping needs
- **Customs Clearance**: Expert customs brokerage and documentation services
- **Logistics Solutions**: End-to-end supply chain management and fulfillment services

### Platform Features
- Real-time shipment tracking
- Quote request system
- Contact forms and inquiry systems
- Responsive design for all devices
- Professional UI with modern design
- SEO optimized

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Custom components
- **Deployment**: Ready for Vercel, Netlify, or any Node.js hosting

## Getting Started

### Prerequisites
- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd china-trade-platform
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```
